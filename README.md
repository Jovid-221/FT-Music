# Asia BG Music

Одностраничный сайт лейбла Asia BG Music.

## Файлы
- `index.html` — весь сайт (HTML/CSS/JS в одном файле, ничего дополнительно подключать не нужно).

## Как открыть локально
Просто открой `index.html` в браузере.

## Публикация через GitHub Pages
1. Зайди в настройки репозитория на GitHub: **Settings → Pages**.
2. В разделе **Source** выбери ветку `main` и папку `/ (root)`.
3. Сохрани — через 1-2 минуты сайт будет доступен по адресу вида
   `https://<username>.github.io/<repo-name>/`.
